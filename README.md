# Crypto Dash

A small crypto dashboard built with React to display live market data and price summaries. Includes search functionality to find specific coins and view their details. Sort coins by market cap, price, and volume. Each coin page has a price history chart to show prices over time. 

The dashboard is responsive and works on both desktop and mobile devices.

Tech stack:
- React
- CSS
- JavaScript / Node.js
- HTML
- CoinGecko API
- React Router
- Tailwindcss
- ChartJS

Deployment:
- Netflify
- Git
- GitHub
