# Crypto Dash

A small crypto dashboard built with React to display live market data and simple price summaries. Includes search functionality to find specific coins and view their details. Sort coins by market cap, price, and volume. The dashboard is responsive and works on both desktop and mobile devices.

Tech stack:
- React
- Fetch API
- CSS
- JavaScript / Node.js
- HTML
- CoinGecko API
- React Router
- Tailwindcss
